# 登录认证

## 用户登录
`
POST /public/v1/user/login
`

<table>
    <tr>
        <th style="width:150px;">请求参数</th>
        <th>必须</th>
        <th>说明</th>
    </tr>
    <tr>
        <td>account</td>
        <td>是</td>
        <td>用户名</td>
    </tr>
    <tr>
        <td>password</td>
        <td>是</td>
        <td>密码</td>
    </tr>
</table>

<table>
    <tr>
        <th style="width:150px;">返回参数</th>
        <th>类型</th>

        <th>备注</th>
    </tr>
    <tr>
        <td>account</td>
        <td>string</td>
        <td>登录用户名</td>
        <td>-</td>
    </tr>
    <tr>
        <td>admin</td>
        <td>bool</td>
        <td>是否管理员</td>
        <td>-</td>
    </tr>
    <tr>
        <td>name</td>
        <td>string</td>
